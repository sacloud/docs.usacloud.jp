    GPUプラン(高火力プラン)対応

    サーバ作成時にGPUプランを指定可能になりました。 usacloud server-plan listコマンドなどでGPUプランの情報を参照し、core、memory、gpuパラメータでサーバプランを指定することで利用可能です。

    Info

    現在はis1aゾーンでのみGPUプランが提供されています

    利用例:

    resource "sakuracloud_server" "foobar" {
  zone = "is1a" 

  name   = "example"

  # 2021/10現在はis1aゾーンにて"GPUプラン/4Core-56GB-1GPU"のみが提供されている
  core   = 4
  memory = 56
  gpu    = 1

  # ...
}
