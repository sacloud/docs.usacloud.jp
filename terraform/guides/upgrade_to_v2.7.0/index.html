    VPCルータのバージョン指定

    VPCルータでバージョン指定が可能になりました。

    利用例:

    resource "sakuracloud_vpc_router" "standard" {
  name      = "standard"

  version = 2
}

    データベースアプライアンスでのパラメータ指定

    データベースアプライアンスでRDBMS固有のパラメータを指定可能になりました。

    利用例:

    resource "sakuracloud_database" "foobar" {
  # 中略...

  parameters = {
    max_connections = 100
  }
}
